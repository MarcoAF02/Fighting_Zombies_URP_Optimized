-----------------------------------------      EFFECTS USING    ----------------------------------------------------------------------------------------------------------
1) You can use each prefab of "muzzle flash" or "impact effect" separately
or 
1) Place "\Assets\Kryptofix\Muzzle Flashes\Prefab\Fire Manager.prefab" to a gun muzzle position.
2) Set the "bullet distance" and "Impact effect prefab"
3) Add for each objects with collision (walls/floor etc) the script "Material Type.cs" and set a type of material (for example, for a brick wall set the "brick")
